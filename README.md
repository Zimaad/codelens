### Deep Code Intelligence • Noir Visual Exploration • AI-Powered Insights

**CodeLens** is a high-performance, AI-native codebase exploration platform built for modern engineering teams. It transforms complex repositories into intuitive, navigable visual landscapes, combining low-latency file exploration with deep LLM-powered intelligence.

---

## ✨ Key Features

- **🌑 Noir Terminal Aesthetic**: A premium, high-contrast dark theme designed for focus and aesthetic excellence.
- **📂 Interactive File Explorer**: High-speed codebase navigation with tree-view hierarchy and contextual file details.
- **🤖 Intelligence Chat Interface**: Integrated AI assistant to query your codebase, understand logic flows, and refactor code on the fly.
- **📊 Dependency Visualization**: Real-time dependency graph rendering using D3.js to map out architecture and identify bottlenecks.
- **⚡ Performance-First Architecture**: Built on **Vite 8** and **React 19** for near-instant rendering and a smooth user experience.
- **🌐 Global Pulse View**: Interactive 3D globe visualization (via Cobe) for a state-of-the-art telemetry feel.

---

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (with advanced HSL color system)
- **Visualization**: [D3.js](https://d3js.org/) & [Cobe](https://github.com/shuding/cobe)
- **Routing**: [React Router 6](https://reactrouter.com/)
- **Animations**: CSS Variables + Tailwind Transitions

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18.0 or higher)
- [NPM](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Zimaad/codelens.git
   cd codelens
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables**
   Create a `.env` file in the root directory:
   ```env
   VITE_AI_API_KEY=your_api_key_here
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
